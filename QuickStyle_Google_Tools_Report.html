Google Agent Development Kit (ADK)
Multi-Agent Orchestration Framework
Free (Open Source) Core
ADK is Google's open-source, production-grade Python/TypeScript/Go framework for building multi-agent AI systems. It handles agent orchestration, tool calls, state management, session memory, and inter-agent communication — all without boilerplate. ADK is optimized for Gemini but supports 100+ models via LiteLLM. Its killer feature is native multi-agent orchestration — agents that delegate subtasks to each other without glue code.
How it powers Quick_Style
  • Orchestrates all 5 agents (Stylist, Route, Onboarding, NEGO BRAIN, Recommendation) as a single composable multi-agent system — agents hand off tasks to each other seamlessly.
  • The Stylist Agent uses ADK's LlmAgent with NL understanding + tool calls to query boutique inventory, filter by size/occasion/budget, and trigger autonomous checkout.
  • SequentialAgent and ParallelAgent patterns handle the boutique onboarding pipeline (photograph → detect → generate image → write description → publish) as parallel steps.
  • ADK's built-in State and Session management powers NEGO BRAIN's multi-turn return conversations with persistent memory of prior interactions.
  • The LoopAgent pattern continuously re-evaluates delivery routes mid-journey when disruptions occur.
  • One-command deployment to Cloud Run or Agent Engine with built-in OpenTelemetry tracing for production observability.
Pricing
Completely free — ADK itself is Apache 2.0 open source. You only pay for the underlying model calls (Gemini API) and the infrastructure you deploy to. No license fees. No per-agent costs.
👗
Vertex AI Virtual Try-On (Imagen)
Image Generation / Fashion AI
$300 GCP Credit Core
Vertex AI's dedicated Virtual Try-On model (virtual-try-on-preview-08-04) is a custom diffusion model that takes two inputs — a person image and a garment image — and generates a photorealistic image of that person wearing that garment. It preserves the person's body shape, pose, lighting, and the garment's texture, folds, and shadows with high fidelity. Updated Sept 2025 for better body preservation. Supports tops, bottoms, one-pieces, and footwear.
How it powers Quick_Style
  • This is your Virtual Try-On feature from Slide 3. Upload user's photo + boutique garment → generate try-on image.
  • Directly attacks the 40% return rate problem at source — "see it on yourself before buying" creates fit confidence.
  • Google reports 60% more high-quality views on Google Shopping for merchants with VTO enabled.
  • Can also be used for boutique catalogue enrichment — generate multiple model shots from a single flat garment photo.
Pricing
Billed under Imagen pricing on Vertex AI. Starts with $300 GCP free trial. Image generation typically $0.0001–$0.002 per image. VTO-specific pricing is under Imagen SKU — use GCP free trial credits to validate at MVP scale.
📸
Imagen 4 (Vertex AI)
Image Generation / Product Photography
$300 GCP Credit Core
Imagen 4 (imagen-4.0-generate-001 and imagen-4.0-fast-generate-001) is Google's highest-fidelity text-to-image and image-editing model. For fashion specifically, it excels at generating studio-quality product images, editing backgrounds, creating flat-lay visuals with consistent lighting, and generating ghost mannequin or virtual model shots. Its "fast" variant handles high-volume catalogue generation economically.
How it powers Quick_Style
  • Powers the Boutique Onboarding Agent — boutique photographs a flat garment, Imagen 4 generates the professional mannequin-framed product image that gets published as the SKU listing.
  • Background removal and neutral-studio replacement for consistent catalogue aesthetics across all boutiques.
  • Imagen 4 is specifically cited for product accuracy and text legibility — critical for preserving garment labels and patterns faithfully.
  • Use Imagen 4 Fast for high-volume onboarding (₹-efficient); use Imagen 4 Ultra for hero product shots and marketing assets.
Pricing
Generally available. imagen-4.0-generate-001 and imagen-4.0-fast-generate-001 are now GA. $300 GCP free trial covers development and early boutique onboarding. Fast variant is significantly cheaper per image.

Gemini Live API (ADK Streaming)
Real-Time Voice + Multimodal AI
Now GA $300 GCP Credit

It supports 70+ languages, natural barge-in interruption, acoustic cue understanding (tone, pace), and real-time tool calling. ADK Streaming wraps it cleanly so you can build voice agents without managing WebSocket infrastructure.

🔍
Vertex AI Multimodal Embeddings + Vector Search
Semantic Search / Recommendations
$300 GCP Credit Core
Vertex AI Multimodal Embeddings convert text, images, and video into aligned vector representations in a shared semantic space. Vector Search (built on Google's ScaNN) indexes these vectors and runs approximate nearest-neighbor queries at massive scale with low latency. Together, they allow "search by image," "text-to-image search," and "find similar items" — the backbone of any modern recommendation system.
How it powers Quick_Style
  • Powers your "Complete the Look" feature — embed a selected blazer, query for style-coherent trousers, shoes, and accessories. The model understands outfit coherence, not just category cross-sells.
  • Enables visual search — user photographs a garment they own or see somewhere, uploads it, and finds similar items across all boutiques.
  • Drives the Stylist Agent's retrieval — NL query → text embedding → vector search → ranked product results aligned with occasion, colour, and body type context.
  • Personalization layer: embed user style history → find products in the same style cluster across new boutiques they haven't discovered yet.
Pricing
Embeddings priced per 1,000 characters (text) or per image. Vector Search index hosting has monthly charges. $300 GCP credit covers initial index build and first months of operation. Very cost-effective at boutique scale (thousands, not millions of SKUs).

Google Maps Routes API
Real-Time Navigation / ETA
5,000/mo Free (Pro SKU)
The Routes API (replacing legacy Directions API) computes optimized routes with live traffic awareness, including TRAFFIC_AWARE and TRAFFIC_AWARE_OPTIMAL modes. It returns turn-by-turn maneuvers, real-time ETAs, alternative route options, and hyperlocal "last-mile" data including building entrances, parking lots, and precise delivery addresses. Supports two-wheel vehicle routing (critical for India's delivery partner fleet).
How it powers Quick_Style
  • The Delivery Route Agent calls Routes API to calculate the optimal path from boutique to customer, accounting for live Delhi/Mumbai traffic in real time.
  • The live map on the User Portal (delivery partner location + ETA countdown) is powered by Routes API waypoint data.
  • Two-wheel routing (Enterprise SKU) is specifically relevant — your delivery partners on bikes and scooters need motorcycle-safe routing, not car routes.
  • TRAFFIC_AWARE_OPTIMAL mode enables dynamic re-routing mid-journey when accidents or traffic jams emerge — core to sustaining the 12-minute SLA consistently.
Pricing (post-March 2025)
5,000 free Pro SKU events/month. Basic route compute: ~$0.005/request. Traffic-aware: slightly higher. Two-wheel routing is Enterprise-tier and billed separately. Maps SDK for Android/iOS: unlimited free.

Google Maps Route Optimization API
Fleet Routing / Multi-Stop Optimization
Paid (Per Shipment)
Route Optimization (formerly Cloud Fleet Routing) solves the Vehicle Routing Problem — assigning multiple delivery tasks across multiple drivers while respecting time windows, vehicle capacities, driver availability, and other constraints. Unlike the basic Routes API (point-to-point), this handles fleet-wide optimization. Skroutz improved on-time delivery from 93% to 98.5% using this API.
How it powers Quick_Style
  • Once you have multiple concurrent orders from the same boutique zone, Route Optimization clusters and batches deliveries for the nearest available partner — reducing total km per delivery.
  • As QuickStyle scales to 50+ boutiques across 2 cities, fleet-wide optimization becomes economically critical — more deliveries per partner hour means lower per-delivery cost.
  • Can handle dynamic re-assignment when a partner cancels mid-delivery or is unexpectedly delayed.
Pricing
Paid per shipment. Single-vehicle routing and fleet routing billed differently. Not needed at MVP stage (1–5 boutiques). Activate when you scale to multi-boutique multi-partner dispatch. Use $300 GCP credit to test.

Firebase (Realtime DB + Firestore + Auth)
Real-Time Data / Authentication
Generous Free Tier
Firebase is a suite of backend services from Google. Firebase Realtime Database pushes JSON state changes to all connected clients instantly. Cloud Firestore is a scalable, queryable NoSQL document store. Firebase Authentication handles user login (Google, phone OTP, email). Firebase Cloud Messaging (FCM) sends push notifications. All have generous free tiers and integrate natively with Cloud Run and Vertex AI.
How it powers Quick_Style
  • Firebase Realtime DB powers the live order tracking on the User Portal — packing status → "Out for delivery" → ETA countdown. Changes pushed to the user's screen in milliseconds.
  • Firestore stores the boutique inventory, order history, user profiles, and return reason analytics. The Boutique Portal's real-time stock levels are kept in sync across all users via Firestore listeners.
  • Firebase Auth handles OTP-based phone login for customers (no email needed — critical for Indian market) and separate admin credentials for boutiques.
  • FCM sends push notifications: "Your order is packed!", "Rider is 2 minutes away!", "Rate your experience" — the full post-delivery engagement loop.
Pricing (Spark Free Plan)
100 concurrent Realtime DB connections, 1GB storage, 50K reads/day on Firestore — free forever. Firebase Auth: 10K SMS verifications/month free. FCM: completely free with no limits. Upgrade to Blaze (pay-as-you-go) only at scale.

Cloud Run
Serverless Container Hosting
2M Requests/mo Free
Cloud Run is Google Cloud's fully managed container execution service. You package your agent backend in a Docker container, deploy it with one command, and Cloud Run handles scaling from zero to thousands of instances automatically. No server management, no Kubernetes complexity. ADK agents deploy to Cloud Run natively with the Agent Starter Pack's one-command deployment.
How it powers Quick_Style
  • Hosts the backend API for all 5 ADK agents — each can be a separate Cloud Run service, independently scaling with demand.
  • The Boutique Onboarding Agent's image-processing pipeline (Gemini vision + Imagen 4) runs as a Cloud Run job — triggered when a boutique uploads a photo.
  • Agent Starter Pack generates the Terraform + Cloud Build CI/CD pipeline for Cloud Run deployment automatically — your infrastructure-as-code is pre-written.
  • Scales to zero when idle (overnight, off-peak) — critical for keeping costs minimal at early-stage operations.
Pricing
2 million requests/month free, plus 360,000 GB-seconds compute and 180,000 vCPU-seconds free every month — the Always Free tier. For a seed-stage startup with 5 boutiques, Cloud Run costs are effectively ₹0/month.

Vertex AI Agent Engine (Runtime)
Managed Agent Runtime
Paid (vCPU Hours) $300 GCP Credit
Vertex AI Agent Engine (renamed from Reasoning Engine / LangChain on Vertex AI) is a fully managed runtime for deploying Python-based agents built with ADK, LangChain, LangGraph, LlamaIndex, or CrewAI. It provides enterprise-grade infrastructure: session state persistence, built-in Memory Bank, A2A (Agent-to-Agent) protocol support, bidirectional streaming, code execution sandbox, and IAM-governed access. Your agent code runs here without you managing compute.
How it powers Quick_Style
  • Deploy all 5 agents to Agent Engine as the managed runtime — no Docker, no Kubernetes, no server management.
  • The built-in Memory Bank is critical for NEGO BRAIN — it persists return history, quality score adjustments, and user pattern data across sessions.
  • A2A protocol allows the Stylist Agent to call the Recommendation Engine agent programmatically as a structured sub-task — clean inter-agent communication without custom glue code.
  • Agent Engine's code execution sandbox (Preview) lets NEGO BRAIN run Python to compute return rate statistics and quality score updates in-flight.
Pricing
Charged per vCPU-hour and GiB-hour since March 2025. Not free, but costs are modest at small agent scale. Consider starting on Cloud Run (free tier) and migrating to Agent Engine once transaction volume justifies managed state persistence.

Secrets / API Key Vault
Free (6 versions / 10K ops)
Secret Manager is a fully managed secret vault that stores API keys, database passwords, and service credentials encrypted at rest, with IAM-governed access, full audit logging, and automatic rotation support. The free tier covers 6 active secret versions and 10,000 access operations per month — more than enough for a startup.
How it powers Quick_Style
  • Securely stores: Razorpay/Stripe payment gateway keys, Google Maps API keys, SMS OTP provider keys, Gemini API keys, Firebase service account credentials.
  • Your ADK agents running on Cloud Run or Agent Engine pull credentials from Secret Manager at runtime — no hardcoded keys in code or environment variables.
  • Audit logging tells you exactly which service read which key at what time — essential for compliance when handling payment data.
Pricing
Permanently free for 6 active secret versions and 10,000 access operations per month. For a startup with <20 secrets queried infrequently, this is ₹0 forever.

Excluded Resources

Tools That Don't Fit (and Why)

Not every tool in the Phase 1–5 resource list adds genuine value to QuickStyle. Here's an honest assessment of what to skip.

Arize (Observability Partner)
Enterprise ML monitoring platform. Adds cost and complexity for a seed-stage startup. Cloud Trace + OpenTelemetry (built into Agent Starter Pack) covers your observability needs at no extra cost.
Fivetran (Data Integration)
A data pipeline tool for moving data between enterprise systems. QuickStyle doesn't yet have the multi-source data warehouse complexity that justifies Fivetran's cost. Use Cloud Functions + Firestore for data movement.
Elastic (Search Partner)
A third-party search engine with significant operational overhead. Vertex AI Vector Search + Agent Builder Data Stores provide equivalent and better-integrated semantic search within the GCP ecosystem — no separate Elastic cluster needed.
Dynatrace (APM Partner)
Enterprise application performance monitoring. Cloud Run's built-in metrics, Cloud Trace, and Cloud Logging provide sufficient observability for a startup. Revisit at 50K+ daily orders.
GitLab (DevOps Partner)
If you're already on GitHub, the Agent Starter Pack generates GitHub Actions CI/CD pipelines natively. No reason to migrate to GitLab CI/CD. If you use GitLab already — then relevant.
Agent Builder Low-Code Path
The managed, no-code Dialogflow CX-based builder is designed for simple Q&A chatbots. QuickStyle's agents require custom tool calling, multi-step reasoning, and inter-agent communication — all of which demand the code-first ADK approach, not the low-code builder.
