─────────────────────────────────────────────
GARMENT-SPECIFIC FIT NOTES
─────────────────────────────────────────────

HOODIES & SWEATSHIRTS:
  - Generally size up for an oversized/streetwear fit
  - True-to-size for athletic/structured look
  - Sleeve length tends to run long in local boutiques

FORMAL SHIRTS & BLAZERS:
  - Size matching is critical — never compromise on shoulder seam alignment
  - Chest circumference is the primary measurement for blazers
  - In local boutiques, L formal shirts often fit like M in international brands

KNIT SWEATERS:
  - Size up by one if wearing layers underneath
  - Merino/wool blends may shrink 5–10% on first wash
  - Shoulder drop is key for comfort

ACTIVEWEAR / TEES:
  - Local boutique activewear runs 1–2 sizes smaller than international brands
  - Always size up for activewear tops if between sizes

─────────────────────────────────────────────
FIT ACCURACY SCORING METHODOLOGY
─────────────────────────────────────────────

QUICK_STYLE's AI assigns a True Fit Score (0–100%) to each garment based on:

1. Brand-to-brand size conversion confidence (35% weight)
2. Garment category norms (25% weight)
3. Fabric composition elasticity factor (20% weight)
4. Customer return rate for that specific garment (20% weight)

Scores above 90% = Highly reliable fit prediction
Scores 75–90%    = Good match, minor adjustments may be needed
Scores below 75% = Recommend consulting size guide or trying in store
